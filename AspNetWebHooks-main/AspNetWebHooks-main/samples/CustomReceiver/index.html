Microsoft ASP.NET WebHooks Custom Receiver

The Custom WebHook Receiver provides support for receiving generic WebHooks generated by the ASP.NET WebHooks Custom WebHooks module.

A sample WebHook URI is:

https://<host>/api/webhooks/incoming/custom/{id}

Set the 'MS_WebHookReceiverSecret_Custom' application setting to the application secrets, optionally using IDs to differentiate between multiple WebHooks, for example 'secret0, id1=secret1, id2=secret2'.
